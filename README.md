# airbnb-data-analysis
An end-to-end Power BI dashboard project built to showcase practical analytics and business intelligence skills. This project emphasizes analytical thinking through data modeling, DAX, Pareto analysis, seasonality trends, pricing strategy, and customer trust insights. More than just building dashboards, it demonstrates the reasoning behind every design choice and highlights why each analytical decision matters.
